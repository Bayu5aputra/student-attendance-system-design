## üöÄ Project Overview

The Student Attendance System is a mobile-first solution designed to replace traditional attendance methods with a more efficient, secure, and reliable process. The application allows students to log in with their university credentials, view their academic dashboard, and record their attendance for classes using GPS validation to ensure they are physically present on campus. This UI/UX case study emphasizes simplicity, functionality, and a modern aesthetic to create an application that students will find easy and enjoyable to use.

## ‚ú® Key Features & Functionality

This application is built around a core set of features designed to enhance the student's academic life:

*   **üîê Secure Authentication:** A simple and secure login screen using official Student ID (NIM) and password.
*   **Dashboard Hub:** A centralized and intuitive dashboard providing quick access to all essential features, including attendance, academic history, campus gallery, and lecturer information.
*   **üìç GPS-Powered Verification:** To prevent academic dishonesty, the app uses GPS technology to verify that the student is within the university campus premises before allowing them to check in.
*   **‚úÖ Instant Confirmation:** Upon successful check-in, a detailed confirmation screen provides immediate feedback, including class details, lecturer name, timestamp, and location coordinates.
*   **üìä Attendance History:** A comprehensive log of all past attendance records, allowing students to track their presence throughout the semester.
*   **Modern & Stylish UI:** A visually appealing design featuring a clean layout, gradient backgrounds, and card-based elements for a modern and modish user experience.
